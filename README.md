# ✅ TODO List

할 일을 그룹별로 정리하고 완료 여부를 관리하는 웹 애플리케이션입니다.

## 기능

- **할 일 추가** — 제목과 설명을 함께 입력, Enter 키 또는 추가 버튼으로 등록
- **설명 펼치기** — 항목 클릭 시 설명을 펼치거나 접을 수 있음
- **완료 체크** — 체크 시 취소선 표시 및 완료 시각(`yyyy-mm-dd hh:mm:ss`) 자동 기록
- **상태 탭 필터** — 전체 / 진행중 / 완료 탭으로 항목 필터링
- **그룹 관리** — 자유롭게 그룹 추가·삭제 (예: Home, Work)
- **그룹 탭 필터** — 그룹별 탭을 눌러 해당 그룹 항목만 보기
- **완료 항목 삭제** — 완료된 항목 일괄 삭제

## 기술 스택

| 분류 | 기술 |
|------|------|
| 마크업 | HTML5 |
| 스타일 | CSS3 (Vanilla) |
| 로직 | JavaScript (Vanilla) |

## 실행 방법

별도 설치 없이 `index.html` 파일을 브라우저에서 바로 열면 됩니다.

```bash
git clone https://github.com/usnoeyeel/todo-list.git
cd todo-list
# index.html을 브라우저로 열기
```

## 파일 구조

```
todo-list/
├── index.html   # 레이아웃 구조
├── style.css    # 스타일
└── script.js    # 기능 로직
```

## 사용 방법

1. **그룹 추가** — 상단 입력창에 그룹 이름 입력 후 `그룹 추가` 클릭
2. **할 일 등록** — 제목 입력 → 설명(선택) → 그룹 선택 → `추가` 클릭
3. **완료 처리** — 항목 왼쪽 원형 체크박스 클릭
4. **필터링** — 상태 탭(전체/진행중/완료) 또는 그룹 탭 클릭
5. **정리** — `완료 항목 삭제` 버튼으로 완료된 항목 일괄 제거
